# KS-Web
Front-end
# 1. Starting

## 1.1. Prerequisites
* NodeJS v6.x:
  * For windows: https://nodejs.org/en/
  * For linux:
```sh  
  curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
  sudo apt-get install -y nodejs
```
  
## 1.2. Running
Run CMD or Terminal and go to where is the project
 * npm install (Just first time.)
 * npm start
